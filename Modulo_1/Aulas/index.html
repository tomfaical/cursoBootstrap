Módulo 1 - Introdução ao Bootstrap

Contexto

Existem 3 tipos de desenvolvimento:

1. Back-end

É a parte visual do site, a parte com a qual o usuário interage;

. Front-end

É a parte funcional do site, a parte por trás do site, que roda todas as funções necessárias;

3. Full-stack

É a integração de todas as partes do site, incluindo o visual, o funcional, e a manipulação e armazenamento de dados.

Um pouco de história

  • Criado em 2011 por Mark Otto e Jacob Thornton (trabalhavam no twitter)
  • Vantagens
    • Responsividade automática --> "coloca infixos em partes do código, o que torna automaticamente responsivo"
    • Componentes prontos --> "não precisa estilizar do zero, o próprio bootstrap já oferece uma série de facilitações"
    • Documentação completa --> "a documentação do BS tem completa na internet, tornando mt facil de aprender"
    • Velocidade no desenvolvimento
    • Gratuito
  • Quais são os materiais necessários para rodar o BS?
    • Navegador atualizado --> Google Chrome
    • Editor de código --> VSCode
    • Internet --> precisa importar o bootstrap do repo online

Existem 2 formas de incluir Bootstrap no seu projeto

Via CDN

Vantagens

  • fácil de configurar
  • Menos peso no servidor
  • Cache do navegador
  • Sempre atualizado

Desvantagens

  • Precisa de internet
  • Controle limitado
  • Dificuldade de customização

Via Download

Sobre infixos

> Infixos são a forma inteligente do Bootstrap de lidar com responsividade. Funciona como o media query (@media), mas se comporta como uma classe

Breakpoints Infixo de classe Dimensões Descrição
1 Extra pequeno None <576px Padrão
2 Pequeno sm (small) >=576px Celular
3 Médio md (medium) >=768px Tablet
4 Grande lg (large) >=992px Laptop
4 Extra Grande xl (extra-large) >=1200px Notebook
5 Extra Extra Grande xxl (...) >=1440px TV 4K
